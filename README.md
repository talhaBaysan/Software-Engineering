# Software-Engineering
 This prject was built using ReactJs and NodeJs. Please make sure to install NodeJs before running the project
******************************************************************************************************
For installing required packages please run:
	cd api
	npm install
Also run: 
	cd client 
	npm install

*******************************************************************************************************
For running frontend:
	cd client
	npm start
*******************************************************************************************************
For running backend:
	cd api
	npm start
********************************************************************************************************
for testing 
	cd api 
	npm test
